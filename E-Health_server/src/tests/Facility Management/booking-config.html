📊 Cấu hình Resolved (Kế thừa: Branch → Global → Default)

Trả về cấu hình cuối cùng có hiệu lực cho chi nhánh. Trường sources cho biết nguồn gốc: branch global default

Chọn chi nhánh và nhấn "📊 Resolved"

📋 Cấu hình Raw (Dữ liệu thô riêng chi nhánh)

Dữ liệu thô lưu trong bảng booking_configurations. Nếu chưa setup → data = null.

Chọn chi nhánh và nhấn "📋 Raw"

✏️ Cập nhật (UPSERT) cấu hình cho chi nhánh

Chỉ gửi field muốn Override. Gửi giá trị null = xóa override (về Global/Default). Trường để trống = không gửi (giữ nguyên).
